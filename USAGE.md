# 如何白嫖机器人账号？/ How to use `HelloTools-bot`

## 方法一

向本仓库提交Issues表明您的意愿, 我们会在审核后给 _您的邮箱_ 发送账号Token

您**必须**按照如下格式提交Issues:

标题 / Title:
 `[申请] %您的用户名% 申请机器人使用权限`

正文 / Description:
```
user: %您的用户名（必填）%
email: %您的邮箱（必填）%
for use: %简要阐明你为什么要使用本机器人（120字以内）%
```

别忘记替换`%%`里的内容！未按要求的Issues将被关闭并拉黑
